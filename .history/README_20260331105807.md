# 🤖 Telegram X Engagement Growth Bot

A high-performance Telegram Mini App Bot that boosts Twitter (X) engagement through gamification, rewards, and automation. Users complete Twitter tasks to earn points, rewards, or tokens, driving viral reach and engagement.

## 🎯 Features

- **🔐 User Authentication**: Telegram login + Twitter OAuth
- **🐦 Task Engine**: Like, retweet, follow, comment tasks with auto-verification
- **🎮 Gamification**: Points, levels, streaks, leaderboard, referrals
- **💰 Reward System**: USDT/token payouts, NFT rewards, airdrops
- **📊 Admin Panel**: Campaign management, analytics, fraud detection
- **⚡ Telegram Mini App**: Modern Web3 dashboard UI
- **🤖 Automation**: Queue system, rate limiting, shadow-ban detection

## 🚀 Quick Start

### 1. Prerequisites

- Node.js 18+
- MongoDB (local or cloud)
- Telegram Bot Token from [@BotFather](https://t.me/botfather)
- Twitter Developer Account with API keys

### 2. Installation

```bash
# Clone or download the project
cd TelegramBot

# Install dependencies
npm install

# Build frontend
npm run build
```

### 3. Environment Setup

```bash
# Copy environment template
cp .env.example .env

# Edit .env with your keys:
# - TELEGRAM_BOT_TOKEN (from @BotFather)
# - TWITTER_API_KEY, TWITTER_API_KEY_SECRET, TWITTER_BEARER_TOKEN
# - MONGODB_URI (local: mongodb://localhost:27017/telegram_x_growth)
# - JWT_SECRET (generate a secure random string)
```

### 4. Start Development

```bash
# Start bot + server + frontend
npm run dev

# Or start production
npm start
```

### 5. Test the Bot

1. Open Telegram and find your bot: `@YourBotName`
2. Send `/start` command
3. Click "Open Mini App" button
4. Test user registration and task completion

## 🔧 Configuration

### Telegram Bot Setup

1. Go to [@BotFather](https://t.me/botfather)
2. Send `/newbot` and follow instructions
3. Copy the token to `.env`

### Twitter API Setup

1. Go to [Twitter Developer Portal](https://developer.twitter.com/)
2. Create a new app
3. Get API Key, API Key Secret, Bearer Token
4. Set callback URL: `http://localhost:3000/auth/twitter/callback`

### Database Setup

```bash
# Local MongoDB
mongod

# Or use MongoDB Atlas (cloud)
# Set MONGODB_URI in .env
```

## 📊 API Endpoints

### Authentication
- `POST /api/auth/telegram` - Telegram login
- `GET /api/auth/twitter` - Twitter OAuth
- `POST /api/auth/twitter/callback` - OAuth callback

### Tasks
- `GET /api/tasks` - Get available tasks
- `POST /api/tasks/:id/complete` - Complete task
- `GET /api/tasks/history` - User task history

### Rewards
- `GET /api/rewards` - Available rewards
- `POST /api/rewards/:id/claim` - Claim reward
- `GET /api/rewards/balance` - User balance

### Admin
- `POST /api/admin/campaigns` - Create campaign
- `GET /api/admin/analytics` - Get analytics
- `GET /api/admin/users` - User management
